# Ares Music Player

Este proyecto fue generado con [Angular CLI](https://github.com/angular/angular-cli) version 17.3.17 y funciona como una aplicacion web construida con Angular y TypeScript.

## Servidor de desarrollo

Ejecutar `npm start` para iniciar un servidor de desarrollo. Luego abrir `http://localhost:4200/`.

## Compilacion

Ejecutar `npm run build` para compilar el proyecto. Los artefactos de compilacion se almacenan en `dist/music-player/browser`.

## Despliegue en Vercel

Usar `npm run build` como comando de compilacion y publicar el contenido generado en `dist/music-player/browser`.
